# **App Name**: Contribution Mint

## Core Features:

- Wallet Connection: Connect to a wallet using RainbowKit.
- Contribution Details Input: Input contributor username, repository URL, and a summary of the contribution.
- NFT Minting: Mint a PoC NFT using the contract at `0x029549CA7179769Bd934B41F4c8F17979A743DE6` on Base Sepolia with the provided details.
- Minting Status: Display a success message upon successful minting.

## Style Guidelines:

- Primary color: Deep violet (#673AB7) to convey creativity and trust.
- Background color: Light lavender (#F3E5F5), a desaturated tint of the primary color, to create a soft and clean backdrop.
- Accent color: Soft blue (#3F51B5), analogous to the violet, used for interactive elements and to draw attention.
- Body and headline font: 'Inter', sans-serif, for a modern and clean interface.
- Use simple, outline-style icons to represent actions and information.
- Center the main content for focus, with a clean, spacious layout to improve readability.
- Use subtle fade-in animations and smooth transitions for a polished user experience.
